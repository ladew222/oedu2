Sa11y
---------------------

INTRODUCTION
------------

Current Maintainer: Bryan Sharpe (https://www.drupal.org/u/b_sharpe)

Sa11y is a client for connecting to an a11y API to check
your sitemap and individual nodes for a11y issues.


INSTALLATION
------------
1. Install the module per normal

2. Configure at /admin/config/services/sa11y

3. Obtain an API key from https://www.sa11y.me/drupal/beta
